PROJECT - Multi-Channel Fashion E-Commerce Sales Analysis (India, 2022)

OVERVIEW 
This project analyzes 31,047 fashion orders generated across six major Indian e-commerce marketplaces—Amazon, Myntra, Flipkart, Ajio, Nalli, and Meesho—between January and December 2022.

BUSINESS QUESTIONS
• Which marketplaces generate the highest revenue?
• Which product categories contribute the most to sales?
• Which customer segments should be prioritized for retention?
• How are sales distributed across regions?
• What opportunities exist to improve long-term revenue growth?


Dataset
• Source: Indian Fashion E-Commerce Sales Dataset (2022)-KAGGLE
• Time Period: January–December 2022
• Records: 31,047 Orders
• Columns: 18
• Unique Customers: 28,437

Tools Used
• Microsoft Excel
• MySQL Workbench
• Power BI

Methodology
1. Cleaned and validated the dataset in Excel.
2. Performed business analysis using SQL in MySQL Workbench.
3. Built an interactive Power BI dashboard.
4. Generated business insights and recommendations.


Key Findings
• Amazon generated the highest order volume.
• Set products contributed nearly 50% of total revenue.
• High Value customers generated more than half of total revenue.
• Monthly revenue remained relatively stable throughout the year.
• Maharashtra, Karnataka, and Uttar Pradesh recorded the highest order volumes.
